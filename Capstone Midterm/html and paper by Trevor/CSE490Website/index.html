Computer Science Major

Course List

CSE-150: Introduction to Computer Science

Introduction to Computer Science showcases the major of computer science here at Midway University by giving broad overviews of the computing field. In this class, you can expect to learn general information about a variety of topics, including algorithms, basic programming, software components, networks, data storage methods such as bytes, and possibly even hardware components. This class is beneficial not only for computer science majors but also for those who wish to learn more about computing, as many of the topics covered are transferable to other majors.

CSE-200: Computer Programming I

Computer Programming I is an introductory course for object-oriented programming languages. Students can expect to learn various programming concepts, data structures, and algorithmic processes, as well as debugging techniques. This course emphasizes structured programming and logical problem-solving, helping students build a coding foundation for future classes.

CSE-210: Computer Programming II

Computer Programming II expands upon the knowledge taken from Computer Programming I, introducing more advanced programming concepts such as object-oriented programming (OOP), polymorphism, complex data structures, and more efficient means of problem solving to further students‚Äô development, allowing them to design larger, modular programs and understand how software systems are structured. Prerequisites include having taken CSE-200 with a minimum grade of C.

CSE-250: Database Systems

Database Systems is a class that examines the theory and architecture of databases. In this class,students will learn to model data, design databases, write and execute queries, and learn SQL. Upon completion, students will have an effective strategy for manipulating data for future applications.

CSE-300: Human Computer Interaction

Human Computer Interaction is a course designed to address interactive applications by applying data-driven approaches to real users. Students can expect to learn about design principles, usability, interface design, and accessibility as they pertain to user experience. The goal of this class is for students to focus on usability and user experience when creating future applications.

CSE-310: Platform Technologies

Platform Technologies will have students focusing on hardware and operating systems for computing platforms. They will learn about operating systems, hardware interaction, virtualization, cloud computing concepts, and system architecture to understand how software interacts with hardware to form the essential components of an IT system.

CSE-320: Networking

Networking covers the principles of computer networking, including network architecture, routing, switching, and network security fundamentals, within a distributed computing setting. Students will learn how data travels across networks and how communication between devices is established, maintained, and debugged.

CSE-400: Cybersecurity

Cybersecurity focuses on one of the most critical areas of technology management. Students can expect to learn about protecting systems, encryption, authentication, risk management, threat analysis, security practices, and ethical applications of topics such as hacking. Theories will be discussed and applied in this course, helping students prepare for careers in cybersecurity after graduation.

CSE-410: Mobile Computing

Mobile Computing is a course designed to teach students how to create and maintain web- and mobile-based applications. Students should be prepared to learn about mobile operating systems, app development frameworks, user interfaces, performance optimization, and additional security concerns across both Android and iOS. Prerequisites include having taken CSE-210 with a minimum grade of C.

CSE-420: Emerging Technologies

Emerging Technologies is designed to allow students to examine new and evolving technologies in both computer science industry and academia. These technologies are ever-changing, so the topics of this course can change each year depending on what‚Äôs gaining importance in the industry, which means that if the topics change, this course is repeatable. As of now, students can look forward to learning about artificial intelligence, machine learning, Internet of Things (IoT), and blockchain.

CSE-490: Capstone

Capstone is a culminating class meant to give students a problem by which they will use knowledge from previous courses to build their solution. Students should be prepared to design, develop, and present a comprehensive project, either in a team setting or alone. This class emphasizes project management, collaboration, documentation, and public speaking, preparing students for post-education workplace settings.

CSE-495: Internship

The Internship course is designed to allow students to practically apply classroom knowledge in a field of their choice within computer science to develop skills for professional growth. Students will have to complete 135 hours of relevant field work at an internship of their choosing, alongside scheduled class meetings. Students will be under the supervision of faculty and site supervisors and are expected to turn in weekly progress reports, along with a final presentation or project, showing the culmination of their work throughout the semester. The internship course is graded pass/fail, restricted to Juniors or Seniors, and may be repeated once.
